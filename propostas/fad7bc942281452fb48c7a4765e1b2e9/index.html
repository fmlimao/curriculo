PROPOSTA DE PRESTAÇÃO DE SERVIÇOS PARA DESENVOLVIMENTO DE SITES E SISTEMAS

APRESENTAÇÃO

Este documento visa formalizar a proposta de prestação de serviços para a criação de um projeto de desenvolvimento de site e sistema para a TLA Produções Artísticas. Dessa forma, descrevemos aqui as principais informações acerca da metodologia de trabalho, do cronograma de atividades, dos recursos necessários para a execução do projeto, bem como as condições de pagamento e prazos previamente definidos.

PRODUTOS/SERVIÇOS

A prestação de serviço consiste no desenvolvimento de um projeto de site e sistema para a TLA Produções Artísticas, que tem por objetivo aumentar o relacionamento com seus clientes e fornecedores, oferecendo mais um canal de comunicação e divulgando os seus serviços através da internet.

Produto: Desenvolvimento do site TLA Produções Artísticas que atenda as seguintes características:

  1. Área institucional: Home, Sobre, Estrutura, Contato, etc (máximo de 10 páginas)
  2. Área de contato: Endereço, telefone, formulário de contato e API Google Maps
  3. Banner rotativo com imagens na página inicial (inclui gerenciador)
  4. Sistema de gerenciamento de conteúdo para galeria de fotos
  5. Sistema de autenticação de usuário (login e senha)

Serviços: Criação da identidade visual, configuração da hospedagem e treinamento.

Produto: Desenvolvimento do sistema TLA - Elenco que atenda as seguintes características:

  1. Módulo de gerenciamento de Elenco, com Perfil interno contendo as informações necessárias para filtros dinâmicos
  2. Módulo de gerenciamento de Clientes e Propostas
  3. Módulo de download de mídias de Elenco e integração dom Dropbox

FASES DO PROJETO

PRIMEIRA FASE: Planejamento e criação

  1. Entrevista com o cliente
  2. Pesquisa e planejamento de projeto
  3. Elaboração de proposta para o projeto

SEGUNDA FASE: Desenvolvimento do Site

  1. Criação da identidade visual (logomarca, layout, fontes, imagens, etc)
  2. Seleção e tratamento das imagens para as páginas e galeria de fotos
  3. Codificação em linguagem programação compatível com os principais navegadores
  4. Desenvolvimento do sistema para o banner rotativo da página inicial
  5. Inserção do conteúdo (textos e imagens) da página inicial e páginas internas
  6. Criação do formulário de contato e direcionamento para em e-mail do cliente

TERCEIRA FASE: Desenvolvimento do Sistema

  1. Criação do painel adminstrativo
  2. Criação dos móludos na seguinte ordem:
    1. Cadastro de Elenco
    2. Filtros dinâmicos
    3. Cadastro de Clientes
    4. Cadastro de Proposta
  3. Codificação em linguagem programação compatível com os principais navegadores

QUARTA FASE: Implantação

  1. Contratação do serviço de hospedagem (custo por conta do cliente)
  2. Compra do domínio e configuração dos DNS (custo por conta do cliente)
  3. Publicação do site na hospedagem contratada
  4. Publicação do sistema na hospedagem contratada

QUINTA FASE: Entrega

  1. Aprovação do cliente
  2. Criação de conta no Google Analytics para monitorar visitantes
  3. Entrega de controle da hospedagem e treinamento básico (2 horas)
  4. Manutenção (correções de erros) até 15 dias após a entrega
